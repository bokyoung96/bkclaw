# Workspace Layout Check

## Goal

Catch accidental root-level clutter before it accumulates again.

## Helper

```bash
python3 scripts/check_workspace_layout.py
```

## Allowed root directories

- `avatars/`
- `bin/`
- `configs/`
- `dags/`
- `docs/`
- `memory/`
- `reports/`
- `scripts/`
- `skills/`
- `src/`
- `tests/`

## Purpose

This is not meant to freeze the repository forever. It exists to make one-off operational files and stray executables visible quickly.
